## Action
- Send fund
- Transfer

## Description
- Send token from my account to another account

## Input
- `network`: Network in list `polkadot`, `kusama`
- `from`: From Account
- `to`: To Account
- `amount`: Amount to send

## Output
transaction in json format
Examples:
```
{
    "network": "polkadot"
    "from": "0x0000",
    "to: "0x00000",
    "amount": 1000000000
}
```
